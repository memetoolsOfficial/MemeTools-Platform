# assets/

Central home for MemeTools' shared, raw source design assets — per
`docs/Phase-0.2-Project-Folder-Structure.md`.

## Contents

- `prototypes/homepage-prototype.png` — the official, approved visual
  specification for the platform's homepage (see `README.md` and
  `docs/Phase-0.4-Design-System-and-UI-Standards.md`). This is the final
  approved design, not a draft or inspiration reference.

## Rules

- This folder holds **raw source assets** only (prototype images, logo
  files, mascot artwork, category iconography source files).
- It never holds compiled/bundled frontend output, user-uploaded content,
  or anything generated at runtime — per Phase 0.2.
- Frontend-build-specific processed assets live in
  `frontend/src/assets/` instead.
